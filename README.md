# -.skill
bat.skill 是一个 GPT Agent Skill，每天帮你发现新 AI 工具，分析它们能干什么、适合谁用、学习成本，并给出行动建议。可选生成气泡图，直观显示工具价值。   bat.skill is a GPT Agent skill that discovers new AI tools daily, analyzes what they do, who they are for, learning cost, and provides actionable recommendations. Optional bubble charts visualize tool value intuitively.
