# bat.skill

## 中文简介

**bat.skill** 是一个面向 GPT Agent 的 AI 工具雷达 Skill。

它的目标是帮助用户跟踪快速变化的 AI 工具生态，每天自动整理你关注领域中的 AI 工具更新，并进一步分析这些工具能做什么、适合谁用、学习成本如何，以及是否值得现在投入时间学习。

## English Introduction

**bat.skill** is an AI tools radar skill designed for GPT Agents.

Its goal is to help users keep up with the fast-changing AI tools ecosystem. It can collect and summarize updates from the AI tool domains you care about, then analyze what each tool does, who it is useful for, how difficult it is to learn, and whether it is worth your time right now.

---

## 核心功能 / Key Features

### 1. 每日 AI 工具更新  
### Daily AI Tool Updates

自动整理你关注领域中的新工具、重要更新和产品变化。

Automatically curates new tools, major updates, and product changes in the AI domains you follow.

---

### 2. 工具用途拆解  
### Use Case Analysis

分析每个工具主要解决什么问题、适合什么人使用、可以用于哪些实际场景。

Explains what problem each tool solves, who it is for, and what real-world scenarios it can be used in.

---

### 3. 学习成本判断  
### Learning Cost Assessment

评估工具的上手难度、学习成本、是否需要立刻学习，避免盲目追热点。

Evaluates how hard each tool is to learn, whether it is worth learning now, and helps users avoid blindly chasing every new trend.

---

### 4. 行动建议  
### Actionable Recommendations

将工具分为：

- 立刻试用
- 收藏观察
- 暂时不用管

Classifies tools into:

- Try now
- Save and watch
- Ignore for now

---

### 5. 气泡图可视化  
### Bubble Chart Visualization

可选生成 X/Y 气泡图，用来直观看出不同工具的价值位置。

Optional X/Y bubble chart visualization to compare tools more intuitively.

示例维度：

- X 轴：学习成本
- Y 轴：实用价值
- 气泡大小：关注度或潜力

Example dimensions:

- X-axis: Learning cost
- Y-axis: Practical value
- Bubble size: Attention or potential

---

## 为什么需要它 / Why This Skill Exists

AI 工具更新太快了。  
很多人不是不想学 AI，而是：

- 不知道哪里有新工具
- 不知道工具到底能干什么
- 学了之后不知道怎么用到自己的工作里
- 容易被热点、榜单和营销话术带着走

AI tools are evolving too fast.  
Many people do not struggle because they are unwilling to learn, but because:

- They do not know where to discover new tools
- They do not understand what the tools actually do
- They do not know how to apply them to their own work
- They are easily distracted by hype, rankings, and marketing language

**bat.skill 的目标不是制造更多信息，而是把 AI 工具信息转化成判断和行动。**

**The goal of bat.skill is not to create more information, but to turn AI tool updates into judgment and action.**
